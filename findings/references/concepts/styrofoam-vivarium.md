## Foam-board / XPS vivarium builds

- [imitation-example] **Pumilo — "Vivarium Built into a Wall" (Dendroboard)** — Long-running build thread documenting an in-wall vivarium using XPS/foam construction with a glass front. Frequently cited reference for foam-board vivarium architecture in the dart-frog community. URL: https://www.dendroboard.com/threads/my-vivarium-built-into-a-wall.63294/
- [imitation-example] **NEHERP — Custom Vivarium Background Tutorial** — New England Herpetoculture's documented technique using XPS panels + Great Stuff + silicone/coco coir for full vivarium interiors. Photographs of actual built large enclosures. URL: https://www.neherpetoculture.com/custombackgrounds
- [imitation-example] **Arachnoboards — "120-gal Bubinga Plywood + XPS Vivarium with Waterfall"** — Documented slow-build of a 4ft+ plywood-shell dart-frog vivarium with XPS foam interior, waterfall, and glass front. URL: https://arachnoboards.com/threads/120-gal-bubinga-plywood-vivarium-build-for-dart-frogs-with-waterfall-slow-build.249771/
