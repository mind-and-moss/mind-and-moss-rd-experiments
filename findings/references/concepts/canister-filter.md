## DIY 5-gallon-bucket canister filter builds

- [imitation-example] **Joey Mullen / The King of DIY — XL aquarium canister filter from a 5-gallon bucket** — The canonical YouTube build the entire DIY-canister scene cites. Two-part build using a Gama Seal lid (the threaded-lid trick is exactly the "0 leaks" lid-sealing problem this concept is solving). URL part 1: https://www.youtube.com/watch?v=hL271pKi_qY — URL part 2: https://www.youtube.com/watch?v=jzJ_QmplCFE
- [imitation-example] **DIY Reptiles — "DIY Canister Filter From A 5 Gallon Bucket!!!"** — Reptile-keeper's YouTube build of a 5-gallon bucket canister, useful as a second perspective on the lid-sealing approach. URL: https://www.youtube.com/watch?v=2Q9dELIPGkQ
- [imitation-example] **YouTube — "DIY 5 Gallon Bucket Canister Filter For 20 Gallon Long Aquascape"** — Build sized for a planted aquascape tank; closer to the boutique aesthetic Isaiah is targeting than utilitarian Joey-style builds. URL: https://www.youtube.com/watch?v=-IT97TH0j_4
