# References — Hardscape Cast Workflow (imitation examples)

Real built work and commercial references for the polymer-clay-over-3D-print → silicone mold → cast hardscape pipeline used in The Gem. Cross-references `products/the-gem/hardscape-workflow.md` and `topics/fabrication/3d-print-prep.md` for the AI-excavation content. See `findings/references/README.md` for tag conventions.

## Polymer-clay → silicone mold → cast hardscape (commercial production)

- [imitation-example] **Universal Rocks** — Commercial maker producing custom-sized 3D aquarium and reptile backgrounds via sculpted masters poured into molds and sprayed with polyurea/polyurethane foam at 160°F. The exact production-scale version of Isaiah's workflow; their material recipe (oxides and sands mixed *before* the pour, not painted on) is worth stealing. URL: https://universalrocks.com/blog/material-options
- [imitation-example] **Aquadecor** — 1,000+ handcrafted background models, custom-sized to the buyer's tank, made by sculpt → mold → cast in polyurethane-family resin. Direct competitor proof-of-concept for one-off castable hardscape as a business. URL: https://aquadecorbackgrounds.com/

## Print-master → urethane-cast tutorials (general fabrication, not aquarium-specific)

- [imitation-example] **Sculpteo — "Urethane Casting from a 3D Printed Master"** — Step-by-step build log: 3D print master → silicone mold → urethane cast, with finished part photos. Not aquarium-specific, but the cleanest documented version of the print-master-then-cast pipeline. Pair it with polymer-clay surface detailing on the print before molding. URL: https://www.sculpteo.com/blog/2016/04/07/diy-tutorial-urethane-casting-with-a-3d-printed-master/
